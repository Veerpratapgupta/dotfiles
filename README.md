# 🎌 Anime Hyprland Dotfiles

> Minimal, fast, anime-aesthetic Hyprland setup for Arch Linux



![Arch Linux](https://img.shields.io/badge/Arch_Linux-1793D1?style=for-the-badge&logo=arch-linux&logoColor=white)




![Hyprland](https://img.shields.io/badge/Hyprland-00A8CC?style=for-the-badge)



## 🖥️ System
- **OS**: Arch Linux
- **WM**: Hyprland
- **Terminal**: Kitty
- **Shell**: Zsh + Starship
- **Bar**: Waybar
- **Launcher**: Wofi
- **Notifications**: Dunst
- **Wallpaper**: Swaybg
- **Lock**: Hyprlock

## ⚡ Performance
- RAM idle: ~400MB
- Faster than HyDE and end-4

## 🎨 Theme
- Catppuccin Mocha colors
- Purple/pink anime aesthetic
- Transparent terminal
- Japanese workspace numbers

## 📦 Install
git clone https://github.com/Veerpratapgupta/dotfiles
cp -r dotfiles/.config/* ~/.config/.
